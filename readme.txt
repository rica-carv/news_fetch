News Fetch Plugin for e107
==========================

Este plugin permite importar automaticamente a primeira notícia de uma lista de URLs, que podem ser feeds RSS ou páginas HTML (scraping).

Funcionalidades:
- Gestão de múltiplas fontes via interface admin
- Importa apenas a primeira notícia de cada fonte, evitando duplicados
- Regista logs de operações e erros
- Fácil de adicionar novas fontes e categorias

Instalação:
- Copie a pasta news_fetch para e107_plugins/
- Instale o plugin via admin do e107
- Configure as fontes no menu "News Fetch"
- Clique em "Importar agora" para executar a importação manual
